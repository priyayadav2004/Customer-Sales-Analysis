# 📊 Customer Sales Analysis Project

## 📌 Project Overview

This project focuses on analyzing customer sales data to uncover meaningful insights and improve business decision-making. The dataset was cleaned, processed, and analyzed using Python, followed by visualization and dashboard creation in Power BI.

---

## 🎯 Objectives

* Clean and preprocess raw sales data
* Handle missing values and outliers
* Perform exploratory data analysis (EDA)
* Generate business insights from customer behavior
* Build an interactive Power BI dashboard

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy)
* Jupyter Notebook
* Power BI
* Git & GitHub

---

## 📂 Dataset

The dataset contains customer-related information such as:

* Customer ID
* Name
* Gender
* Age
* Location
* Purchase Amount
* Feedback Score

---

## 🔍 Data Cleaning & Processing

* Removed negative and inconsistent values
* Handled missing data
* Detected and removed outliers using Z-score
* Standardized and formatted data for analysis

---

## 📊 Exploratory Data Analysis (EDA)

* Age distribution of customers
* Purchase amount trends
* Customer segmentation
* Feedback score analysis

---

## 📈 Key Insights

* Certain age groups contribute more to revenue
* Outliers significantly impacted purchase analysis
* Customer feedback correlates with purchase behavior
